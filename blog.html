What is DOM? What is the purpose of DOM?

DOM means Document Object Model. When the browser constructs the Document Object Model, it goes through five different steps. At the end of the fifth step, you get the DOM.

How to get select element from DOM?

Six methods to select an element from the DOM.

  • 1. getElementById
  • 2. getElementsByClassName
  • 3. getElementsByName
  • 4. getElementsByTagName
  • 5. querySelectorAll
  • 6. querySelector

What is Event Bubble?

Event bubbling is the propagation of events from the bottom to the top of the DOM tree, from child elements to parent elements.

What is Callback Function? How it is used?

A callback is a function passed as an argument to another function. This technique allows a function to call another function. A callback function can run after another function has finished.
